Protify

A simple, client-side profanity filter for your web projects.

What is Protify?

Protify is a straightforward, free-to-use profanity filtering solution that runs entirely in the user's browser. It works by fetching a comprehensive JSON list of "bad words" and using a small JavaScript function to censor matching words from any text on your website. It's perfect for projects where you need to moderate user-generated content without implementing a complex server-side backend.

How It Works

The process is simple:

  1. Your website's JavaScript fetches the `protify-list.json` file when the page loads.
  2. The list of words is stored in a variable in your script.
  3. A JavaScript function, `censorText()`, takes any string of text as input.
  4. The function compares each word in the text against the loaded word list and replaces any matches with asterisks (e.g., `****`).
  5. The "clean" text is returned and can then be safely displayed on your page.

Implementation Guide

Step 1: Get the Wordlist

You need the `protify-list.json` file. You can download it from the link below or copy the contents. You should host this file on your own server or within your project's repository (e.g., on GitHub Pages, Vercel, or Netlify).

Download protify-list.json

Step 2: Add the JavaScript to Your Site

Place the following JavaScript code in your main script file or within a script tag in your HTML. This code will fetch the list and provide the censoring function.


// Global variable to hold the list of words
let badWordsList = [];

/**
 * Censors text by replacing words from a pre-fetched list with asterisks.
 * @param {string} text The input text to censor.
 * @returns {string} The censored text.
 */
function censorText(text) {
    // Return original text if it's empty or the list hasn't loaded
    if (!text || !text.trim() || badWordsList.length === 0) {
        return text;
    }
    // Create a regular expression for whole-word, case-insensitive matching
    const regex = new RegExp(`\\b(${badWordsList.join('|')})\\b`, 'gi');
    return text.replace(regex, match => '*'.repeat(match.length));
}

// Fetch the profanity list when the page loads
document.addEventListener('DOMContentLoaded', () => {
    // IMPORTANT: Replace this URL with the URL where YOU are hosting the JSON file.
    const jsonListUrl = 'https://4simpleproblems.github.io/Protify/protify-list.json';

    fetch(jsonListUrl)
        .then(response => {
            if (!response.ok) {
                throw new Error('Network response for wordlist was not ok');
            }
            return response.json();
        })
        .then(data => {
            if (data && data.words) {
                badWordsList = data.words;
                console.log('Protify profanity list loaded successfully.');
                // You can optionally run a function here to censor existing content
            }
        })
        .catch(error => {
            console.error('Failed to load Protify profanity list:', error);
            // The app will continue to function, just without censoring.
        });
});

Step 3: Use the Function

Now you can use the censorText() function anywhere in your code to clean text before displaying it.


// Example: Censoring a user comment before displaying it

// Get the user's raw input
const rawComment = "This is some bullshit that should be censored.";

// Censor the text using the function
const cleanComment = censorText(rawComment);

// Display the clean text on your page
const commentElement = document.getElementById('user-comment-display');
commentElement.textContent = cleanComment;

// The output will be: "This is some ******** that should be censored."
